# Reymon Choudhary — Portfolio Website


A responsive and interactive personal portfolio website showcasing my experience, skills, projects, and achievements as a Software Engineer.

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Features

* Responsive design
* Multiple theme options
* Interactive animations
* Typewriter effect
* Canvas particle background
* Smooth scrolling navigation
